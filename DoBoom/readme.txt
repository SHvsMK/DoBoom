The control for our game is the keyborad.
Our game support two players.

For player1:
w: move up
s: move down
a: move left
d: move right
table: put boom
q: tick the boom
leftshift: triger
1: create object 1
2: create object 2
3: create object 3
4: create object 4
5: create object 5
6: create object 6

For player 2:
Arrow up: move up
Arrow down: move down
Arrow left: move left
Arrow right: move right
space: put boom
rightshift: tick the boom
enter: triger
f: create object 1
g: create object 2
h: create object 3
j: create object 4
k: create object 5
l: create object 6

In our game, there are some instructions about the controller, and you can change it in the way you wish.

Some display of game in windows version may look not good. We arrange all the display part in mac os.
If you use 1440 * 900, it will have a good looking.
